🤖 CONFIGURACIÓN RÁPIDA DE CHATGPT
=====================================

PASO 1: Obtener API Key
-----------------------
1. Ve a: https://platform.openai.com/api-keys
2. Inicia sesión o crea cuenta
3. Clic en "Create new secret key"
4. Copia la key (empieza con sk-...)

PASO 2: Configurar en Proyecto
------------------------------
1. Abre el archivo .env en la raíz del proyecto
2. Busca esta línea:
   REACT_APP_OPENAI_API_KEY=your_openai_api_key_here
3. Reemplaza "your_openai_api_key_here" con tu API key real
4. Guarda el archivo

PASO 3: Reiniciar Servidor
--------------------------
1. En la terminal, presiona Ctrl+C para detener
2. Ejecuta: npm start
3. ¡ChatGPT se activará automáticamente!

VERIFICACIÓN
------------
- En el Chat IA verás "POWERED BY CHATGPT ✅ ACTIVO"
- En el sidebar aparecerá punto verde + "ChatGPT"
- Las respuestas mostrarán "RESPUESTA GENERADA POR CHATGPT"
